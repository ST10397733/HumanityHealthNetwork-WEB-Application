# HumanityHealthNetwork-WEB-Application
The humanitiesHealthNetwork website is a static website developed using HTML and CSS. the purpose of this website is to provide users with information related to health wellness and community support. This project focuses on building a clean and structured website layout while applying core front end development skills such as sematic HTML structure, responsive styling and user friendly navigation. this project was developed using Notepad++ as the code editor to strengthen fundamental web development skills and demonstrate an understanding of how websites are built from the ground up.

it featured informational health and wellness content, structured multipage website layout, navigation between pages,styled interface using CSS, clean and readable design and many more.

Tools used: HTML , CSS, NOTEPAD++

Installation: Download the repository, open project folder , and open the index html using any web browser.
